# CS361 Lab 5: Personal Portfolio Website

A static personal portfolio website built using HTML and CSS. This project was developed to demonstrate web development fundamentals, including semantic HTML structure, CSS styling, and layout management.

## Live Demo
[View the live website here](https://jameschang352.github.io/cs361-lab5-portfolio/) 

## Technologies Used
* **HTML5:** Page structure, semantic tags, tables, and lists.
* **CSS3:** Custom styling, color palettes, and image formatting.

## Features Showcase
* **About Me:** Profile picture and introduction.
* **Interests:** Styled list of personal hobbies.
* **Academic Projects:** Structured table showcasing my development work and research.
